Steam Web Project

For this project, my team and I developed an E-commerce platform inspired by Steam, enhancing its visual design and functionalities. My role encompassed both front-end and back-end development, ensuring a seamless user experience and robust API performance. On the front-end, we leveraged Angular, Tailwind, and Bootstrap to create a modern and responsive UI. On the back-end, we built a scalable REST API using Node.js and Express, fully implemented in TypeScript. The database was designed using MongoDB, where we optimized data retrieval with search indexes and hierarchical category structures to enable efficient product navigation. This project was an excellent opportunity to work on full-stack development, combining performance optimization with an intuitive and user-friendly interface.
